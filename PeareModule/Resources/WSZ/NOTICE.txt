WSZ format support in Peare
===========================

The WSZ binary layout and parser behavior used by WszDecoder.cs are based on:

  WMP WSZ Format
  Copyright 2017-2024 Tim De Baets
  https://github.com/tdebaets/wmp-wsz-format

The referenced work is licensed under the Apache License, Version 2.0.
A copy of that license is included as LICENSE-Apache-2.0.txt.

WszDecoder.cs is a C# implementation written for Peare. It does not copy the
Delphi source verbatim and does not require Windows Media Player, its COM type
library, or registry data.
